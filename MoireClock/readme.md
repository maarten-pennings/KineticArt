# Moiré clock

Kinetic Art. 

Two rotating discs with a special pattern cause a Moiré effect.

Inspiration from [https://www.youtube.com/watch?v=--O9eyKIubY&t=24s](https://www.youtube.com/watch?v=--O9eyKIubY&t=24s​).
